# Insulin Reminder
Aplicativo web criado para uso e fins pessoais. Todo o design foi criado em conformidade com o que foi solicitado pela pessoa que vai utilizar.
## Objetivo
A usuária vai logar no aplicativo e preencher um pequeno formulário com seu telefone, a dosagem e o horário que quer receber o lembrete. Daí em diante diariamente ela irá receber no Whatsapp o lembrete.

## Overview técnico
A aplicação foi feita utilizando **NextJs**. Toda a lógica de armazenamento, e disparo do evento foi feita utilizando o ecossistema AWS.

No repositório há um arquivo detalhando as configurações realizadas e o fluxo que ocorre na AWS.
